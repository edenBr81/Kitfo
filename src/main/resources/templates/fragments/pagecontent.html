Fragments
Where does this content come from?

There is a folder called 'fragments' in the templates folder. This contains the base for the navigation. Fragments can be used from ANY html file.

Have a look at how the navbase file is strucutred.

By using the syntax th:fragment="thenav(menuoption)", you are establishing a fragment that will accept a 'menuoption' variable

Depending on the value of this variable, you can add styling. See the 'Home' and 'Link' options in the navbar

The styles that are applied will depend on your template, so make sure you know how the active links work for the template you are using.

Once you find the source fo this text, you are ready to proceed!

Remember: Use fragments wisely. Sometimes you don't need to put text in a fragment - when you're only going to use it once. This text could just have easily gone onto the firstlink page. But when you want to re-use segments of code, fragments are the best way to do so in html.
